\# Resume Builder (Django Project)



This is a web-based Resume Builder built using Django.



\## Features

\- User Registration \& Login

\- Add Personal Details

\- Add Education, Skills, Projects

\- Select Resume Templates

\- Generate Resume



\## Tech Stack

\- Python (Django)

\- HTML, CSS

\- SQLite



\## How to run

1\. Clone repo

2\. cd resume\_builder

3\. python manage.py runserver
